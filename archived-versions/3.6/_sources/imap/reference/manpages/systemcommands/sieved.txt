==========
**sieved**
==========

Script to decompile a sieve script back from bytecode.

Synopsis
========

.. parsed-literal::

    **sieved** *scriptfile*

Description
===========

**sieved** decompiles the given script at *scriptfile* from bytecode, writing output to stdout.
